A collaborative dictionary game for researchers.

About Lexicon

Lexicon is a game in which an arbitrary number of players pretend to be researchers compiling a dictionary. In each round every player will choose an existing, undefined word starting with the next letter in alphabetical order or (if there are none) create a new word starting with that letter and write a definition for it. This definition will include two new words that start with later letters (if there are any).
